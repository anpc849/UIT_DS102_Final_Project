cleaning_data.py (kết hợp với hàm clean_text trong train_model.py): dùng để xử lý raw_data, kết quả thu được sau khi xử lý là một dataset phù hợp để chạy các tác vụ về sau.

training_model_final.ipynb: dùng để tinh chỉnh tham số model và lưu model với các bộ siêu tham số tối ưu bằng thư viện joblib.

Webapp_Sentiment_TopicModel.ipynb: dùng để load các model và triển khai một ứng dụng webapp đơn giản về phân tích cảm xúc và phân tích chủ đề.
